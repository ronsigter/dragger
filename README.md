### Libraries

- [x] React-hook-form for form management
- [x] Jest with React-Testing-Library for unit/integration tests

## Getting Started

1. Install Dependencies

```bash
yarn install
```

2. Start development

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

3. Run Tests

```bash
yarn test
```
